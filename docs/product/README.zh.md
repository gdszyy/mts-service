# 产品文档

本目录包含所有与 `mts-service` 产品设计相关的文档，主要聚焦于用户交互逻辑和核心功能协议。

## 文档列表

- **[前端交互设计](./Frontend_Interaction_Design.zh.md)**：详细阐述了投注平台的前端交互逻辑，包括单关/串关模式、新手/老手模式、异常处理等高级功能。

- **[WebSocket 协议](./WebSocket_Protocol.zh.md)**：定义了前端与 `mts-service` 后台进行实时通信的 WebSocket 协议，用于处理投注延迟问题。

- **[流程图](./flowcharts/)**：包含所有产品设计相关的流程图，使用 Mermaid 格式编写。
